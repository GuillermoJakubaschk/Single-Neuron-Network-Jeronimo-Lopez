# README

Este notebook muestra un ejemplo de una red neuronal simple y compara los resultados usando los datos originales y los comparamos usando PCA en el mismo modelo.

## Requisitos

Necesitas tener Python instalado junto con las siguientes librerías:

* numpy
* pandas
* matplotlib
* scikit-learn
* jupyter

Se pueden instalarlar con:

pip install numpy pandas matplotlib scikit-learn jupyter

## Cómo ejecutar el notebook

1. Descarga el archivo.
2. Abre una terminal en la carpeta donde está el archivo.
3. Ejecutas el programa por partes.
